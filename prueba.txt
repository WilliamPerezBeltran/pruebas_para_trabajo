Hola William,

La prueba consiste en la construcción de un aplicación de efemérides.

La efeméride debe mostrar:

1.- Fecha de efeméride
2.- Categoría o tipo de efeméride (musical, deportiva, noticiosa, etc)
3.- Texto de la efeméride
4.- Imagen asociada a la efeméride

Tome en cuenta los siguientes detalles:

a) Todos los elementos incluyendo las categorías deben ser editables, use elementos como "select" en las vistas Html
b) Debe haber una opción que permita mostrar las efemérides de un día en particular, por defecto debe mostrar las efeméride del día actual
c) Incluya una opción de búsqueda por palabra clave
d) Todos los resultados  se deben mostrar via web o vía Api Rest, a través de una respuesta tipo JSON.

Tienes dos (2) horas a partir de la hora de envío del correo para realizar la prueba. Sino completa la prueba envíe el código generado hasta el momento.

Suerte en la Prueba!
